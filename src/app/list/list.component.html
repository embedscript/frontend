Social frontend development

Embedscript is a tiny framework, a package manager, Firebase-like APIs and a webhost all in one.
Building frontend widgets and sharing them with your friends has never been this easy.

What if Github gists would be alive? With Embedscript writing frontend apps and widgets that can be reused by you
or anyone else is as easy as saving a gist. You will need no backend, as we provide





Write code 
Embed.post("datastore/create", {
  "project":  "projectid-1337",
  "table":    "comments",
  "value":    `"name":"joe", "comment": "Nice article!"`
});
Place embed code to your site 
<script src="https://embedscript.com/e.js"
  embed="name-of-script"
  project="projectid-1337">
</script>

Ready to use scripts, with your own your data

Every script can be instantly used by logged in users.
When you embed the script, it will be done with your own project id.
This means you own all the data that script is accessing, and not the author of the script.

APIs at your fingertips.

Define data access rules for your tables and save data straight from the browser.
Use a users service or streaming to build your next app..
